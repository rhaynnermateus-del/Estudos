Comandos Essenciais de Movimentação
Mover para Cima: Tecla W / Console: ↑
Mover para Baixo: Tecla S / Console: ↓
Mover para Esquerda: Tecla A / Console: ←
Mover para Direita: Tecla D / Console: →
Pular / Confirmar: Tecla Barra de Espaço ou Enter / Console: ✕ (X)
Ação Secundária / Cancelar: Tecla CTRL / Console: ○ (Bola/Círculo)
Menu / Opções: Tecla ESC / Console: △ (Triângulo)
Inventário: Tecla I / Console: □ (Quadrado)
Sequência de Ataque Especial (Combo "Fúria Pixelada")
Pressione □, □ (duas vezes rapidamente)
Em seguida, pressione ○ com precisão
Finalize segurando R2 e apertando ✕
Glossário de Itens do Jogo

Escudo Defletor Mk.II 🛡️
