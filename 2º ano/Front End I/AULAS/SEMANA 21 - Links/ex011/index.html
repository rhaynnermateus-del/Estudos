Links locais, externos e download de arquivos

Os links ou hiperlinks são ligações que existem entre páginas. Vamos ver alguns exemplos do uso das âncoras.

Os links locais apontam para documentos que estão no seu próprio site. outra pagina nesse mesmo servidor clicando no link anterior.

Também podemos colocar documentos em outras pastas locais. Acesse uma página em outra pasta clicando no link anterior.

Também é possível acessar outros sites usando âncoras. Por exemplo, você pode criar um link para acessar o YouTube (não se esqueça de clicar no botão "VOLTAR" do seu nevegador depois, para voltar pra cá)

O link acima faz o usuário ir pra outro site e abandona o meu. O ideal é que ele abra outros sites em uma nova aba. Acesse agora o YouTube em uma nova aba em branco.

Os links também podem ser usados para fazer o download de arquivos de outros formatos como ZIP, PDF, RAR, etc. Por exemplo, abra aqui um livro PDF de exemplo ou abaixe a versão compactada do livro.

OBS:Clique sempre em exibir o código fonte para ver como tudo está sendo criado.
